# 独立時計師養成システム — Cal.02 Automatic

自動巻ムーブメント(Cal.02)を一から組み立てる、スイス時計学校レベルのデジタル実習台です。
サーバー不要の HTML / CSS / JavaScript のみで動作し、外部ライブラリは Three.js だけを使用しています。

## 全体構成(3章)

1. **第1章 ムーブメント組立(機械側)** — 地板 → 香箱 → 二〜四番車 → ガンギ車 → 各受け → アンクル → テンプ → 巻上機構 → 自動巻機構 → ローター(最後)。完成後、ローターでゼンマイを巻くとテンプが振動し輪列が回転します。
2. **第2章 文字盤側組立** — ムーブメントが180°ゆっくり反転し文字盤側が現れる映像演出のあと、モーションワーク(筒カナ・日の裏車・時車)→ 文字盤 → 短針・長針・秒針を取り付けます。
3. **第3章 ケーシング** — ケース・ムーブメントリング・パッキン・風防・ベゼル・裏蓋・竜頭を組み、時計ヘッド単体として完成させます（ベルト無し）。

最後に「Cal.02 を起動」で、高級時計ブランドのCMのような完成演出(約11秒)が流れます。
裏側(ローター)→ 文字盤側へカメラが回り込み、静止して "Cal.02 Automatic" が表示されます。

## 使い方

- 画面下の**部品カード**を3Dビューへドラッグして配置します。正しい部品を正しい位置へ近づけると吸い付くように収まります。
- **Learning Mode**: 次に置く部品が青く発光し、左パネルに「役割 / 他部品との関係 / 解説 / 豆知識」が表示されます。間違えると「なぜ違うのか」を説明します。
- **Exam Mode**: ガイド・解説なし。知識だけで組み立てます。
- カードや配置済み部品をクリックすると部品情報を確認できます。
- 進捗は自動保存されます(右上 RESET でやり直し)。

## 操作

- ドラッグ: ムーブメントを回転 / ホイール・ピンチ: ズーム / カードをドラッグ: 部品配置

## GitHub Pages へのデプロイ

このフォルダをリポジトリに push し、Settings → Pages → Branch: `main` / root を選ぶだけで公開できます。
ローカルで開く場合は JSON を読み込むため簡易サーバー経由でアクセスしてください:

```
python3 -m http.server
```

## フォルダ構成

```
index.html            エントリーポイント
css/style.css         ガラスUI・黒基調・シャンパンゴールド
js/main.js            起動処理(読み込むJSONを MOVEMENT_URL で指定)
js/scene.js           Three.js シーン / 撮影ライティング / 環境反射 / 操作
js/parts.js           部品ジオメトリ + 仕上げテクスチャ(ペルラージュ/コート・ド・ジュネーブ 等)
js/ui.js              DOM UI(章表示・サポート・トレイ・シネマ演出)
js/assembly.js        組立ロジック / 章進行 / 反転・完成シネマティック / 駆動
movement/cal02.json   ムーブメント定義(全35部品・章・座標・順番・解説・関係・誤置理由)
assets/               画像・音声用(将来拡張)
```

## 拡張方法

`movement/cal02.json` と同じ形式で別ムーブメントを作り、`js/main.js` の `MOVEMENT_URL` を
差し替えれば別キャリバーに対応できます。部品は `chapter`(movement / dial / case)・`order`・
`position`・`rotationY`・`run`(駆動定義)・各解説フィールドを持ちます。
新しい部品形状を足す場合は `js/parts.js` の `BUILDERS` にビルダー関数を追加してください。
